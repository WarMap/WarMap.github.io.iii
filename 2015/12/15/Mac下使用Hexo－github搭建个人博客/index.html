Mac下使用Hexo－github搭建个人博客

前言

        先前查资料或问题总能进到他人的blog，心里很是长草。周末决定自己也搞一个,本来做好了万难的准备（以为要自己写界面干啥干啥的，最后还要申请个域名、空间什么的，balabala）后来发现了Hexo和github的组合,简直容易到爆啊，对了做到容易到爆当然少不了homebrew这等神器。好下面开始。对了忘了告诉大家，这篇文章只是写给自己留个记录的，如果你也想通过hexo搭blog请移步到hexo官网

开搞

what do we need？

  • homebrew （从此安装只需一句话）
  • nodejs（hexo基于nodejs）
  • hexo
  • git（提交部署github）

how to do it

安装

  1. 安装Homebrew（一句话） 官网

    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

  2. 安装nodejs（又是一句话）

    brew install node

  3. 安装hexo（又是三句话）（咦，总感觉哪里不对）

    npm install -g hexo
hexo init
npm install

  4. 安装git（还是一句话，就是这个feel倍爽）

    sudo brew install git

配置

  1. 生成SSH keys(Generating SSH keys)

    SSH keys均在用户根目录下.ssh文件夹中保存，会存在之前已经存在key的情况，下面分开讨论

    • 没有key的情况（一路回车在~/.ssh目录下得到id_rsa和id_rsa.pub两个文件）

      ssh-keygen -t rsa -C "注册github的邮箱”

    • 有key的情况（生成key并制定key的名字）

      ssh-keygen -t rsa -f ~/.ssh/key的名字 -C "注册github的邮箱"

  2. 添加key到ssh-agent

    ssh-add ~/.ssh/id_rsa

  3. 添加key到github

    • 打开github，在右上角点击你的头像，然后点击setting
    • 设置的侧边栏点击SSH keys
    • 点击Add SSH key
    • title随便写，把你的key粘贴到下面。然后点击Add key

    • 检查是否关联成功

      ssh -T git@github.com
