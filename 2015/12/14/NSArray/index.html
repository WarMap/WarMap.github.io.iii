NSArray

- (NSArray<ObjectType> *)sortedArrayUsingComparator:(NSComparator)cmptr

Returns an array that lists the receiving array’s elements in ascending order, as determined by the comparison method specified by a given NSComparator Block.
返回一个“升序”排序后的数组，这个“升序”由NSComparator的返回值决定。code give the answer

NSArray *arrar = @[@(1),@(2)];
NSArray *b = [[NSArray alloc] init];
b = [arrar sortedArrayUsingComparator:^NSComparisonResult(NSNumber *obj1, NSNumber *obj2) {
    if (obj1.intValue > obj2.intValue) {
        return NSOrderedDescending;
    }else if (obj2.intValue > obj1.intValue) {
        return NSOrderedAscending;
    }else
        return NSOrderedSame;
}];
NSLog(@"%@",b);


NSArray *arrar = @[@(1)@(2)];
NSArray *b = [[NSArray alloc] init];
b = [arrar sortedArrayUsingComparator:^NSComparisonResult(NSNumber *obj1, NSNumber *obj2) {
    if (obj1.intValue > obj2.intValue) {
        return NSOrderedAscending;
    }else if (obj2.intValue > obj1.intValue) {
        return NSOrderedDescending;
    }else
        return NSOrderedSame;
}];
NSLog(@"%@",b);


由上面可以看出，再cmptr中决定排序要比较的内容和规则。
